## DOM(Document Object Model)
解析器的输出“解析树”是由 DOM 元素和属性节点构成的树结构。     

它是 HTML 文档的对象表示，同时是外部内容（例如 JavaScript）与 HTML 元素之间的接口。    

解析树的根节点是“Document”对象。

DOM 与标记之间几乎是一一对应的关系。比如：   

```html
<html>
  <body>
    <p>
      Hello World
    </p>
    <div> <img src="example.png"/></div>
  </body>
</html>
```
